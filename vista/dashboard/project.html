Resumen Ejecutivo

ViuActiu es una herramienta digital innovadora que utiliza exclusivamente datos abiertos oficiales para facilitar el acceso a recursos de envejecimiento activo y saludable en Catalunya. La plataforma centraliza información dispersa en múltiples fuentes gubernamentales y la presenta de forma accesible, interactiva y útil para ciudadanos, familias, administraciones públicas e investigadores.

Viu Actiu parece un portal estático o semi-interactivo para "gent gran" (personas mayores), con secciones como la "Sobre el Proyecto" que mencionas (perfecta para el formulario del concurso). Probablemente incluye info sobre actividades, eventos y tips para envejecimiento activo en entornos catalanes. Pero para el reto, necesita un "núcleo" de datos abiertos: visualizaciones dinámicas que muestren, por ejemplo, tasas de participación en ocio por edad/sexo/municipio, o mapas de accesibilidad en espacios públicos.

Por qué encaja: Se alinea con ámbitos como participación social, ocio, salud y brecha digital. Podemos transformarlo en una herramienta como las ejemplos del reto: visualizaciones interactivas (tipo DataLab Mayores) que usen datos públicos para impactar en políticas locales.

Problema que resuelve

  • • Información fragmentada en múltiples portales oficiales
  • • Dificultad para localizar servicios geográficamente
  • • Falta de visualizaciones comprensibles de datos complejos
  • • Acceso limitado para personas con necesidades especiales

Solución propuesta

  • • Centralización de 4 fuentes de datos abiertos oficiales
  • • Mapa interactivo con 1.847 centros georreferenciados
  • • Visualizaciones de datos con perspectiva de género
  • • Diseño accesible WCAG AAA y multiidioma

Metodología de Desarrollo

1. Recopilación y Selección de Datos

El proceso de selección de fuentes de datos se basó en criterios estrictos de calidad, actualización y relevancia:

Criterios de Selección

  • Oficialidad: Solo fuentes gubernamentales verificadas (Generalitat, Diputación, CIS)
  • Licencia abierta: CC BY 4.0 o similar que permita reutilización
  • Actualización: Frecuencia mínima trimestral o según disponibilidad oficial
  • Calidad: Datos estructurados, completos y con metadatos descriptivos
  • Relevancia: Directamente relacionados con envejecimiento activo y saludable
  • Desagregación: Preferencia por datos desagregados por sexo, edad y territorio

2. Procesamiento y Transformación de Datos (ETL)

Extracción (Extract)

Conexión automática a APIs oficiales mediante peticiones HTTP. Implementación de reintentos automáticos y manejo de errores para garantizar la disponibilidad del servicio.

Node.js Fetch API Error Handling

Transformación (Transform)

Normalización y limpieza de datos:

  • • Eliminación de duplicados mediante verificación de IDs únicos
  • • Normalización de nombres de campos entre diferentes fuentes
  • • Geocodificación y validación de coordenadas geográficas
  • • Estandarización de formatos de fecha y texto
  • • Cálculo de campos derivados (porcentajes, totales, etc.)

Carga (Load)

Servicio de datos a través de endpoints REST propios que agregan y filtran información de múltiples fuentes. Implementación de caché para optimizar rendimiento.

Express.js REST API JSON Response

3. Validación de Calidad

Proceso riguroso de validación para garantizar la fiabilidad de los datos:

  • ✓ Validación de integridad: Verificación de campos obligatorios y formatos correctos
  • ✓ Coherencia territorial: Comprobación de que comarcas y municipios existen en Catalunya
  • ✓ Validación geográfica: Coordenadas dentro del territorio catalán
  • ✓ Consistencia temporal: Fechas válidas y coherentes
  • ✓ Auditoría de cambios: Registro de actualizaciones y modificaciones

Casos de Éxito Potenciales

Caso 1: Familia en situación de urgencia

Situación: María, de Tarragona, necesita urgentemente una residencia para su madre de 85 años con movilidad reducida.

Solución con ViuActiu: En 15 minutos localiza en el mapa 23 residencias en Tarragona, filtra por plazas disponibles, obtiene teléfonos de contacto y puede comparar servicios. Anteriormente habría necesitado llamar a múltiples organismos durante días.

Caso 2: Ayuntamiento planificando recursos

Situación: El Ayuntamiento de Girona quiere ampliar la oferta de centros de día para personas mayores.

Solución con ViuActiu: Mediante las estadísticas territoriales identifica que su comarca tiene 15% menos centros por habitante que la media catalana. Detecta barrios específicos sin cobertura y puede fundamentar la inversión con datos objetivos.

Caso 3: Investigación sobre feminización del envejecimiento

Situación: Una ONG quiere estudiar las diferencias de género en el acceso a servicios sociales.

Solución con ViuActiu: Accede a datos desagregados por sexo de 145.280 personas atendidas, identifica que las mujeres representan el 64,2% y puede cruzar estos datos con información sobre soledad para diseñar programas específicos.
